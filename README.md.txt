# Zeni – Simple Debt Tracker

Zeni is a clean, simple debt tracking app designed to help people become debt-free without feeling overwhelmed.

Most finance apps feel cluttered or complicated. Zeni focuses on clarity, simplicity, and motivation.

---

## 🚀 Live Demo

https://zenidebtfree.com

---

## ✨ Features

- Track multiple debts in one place
- Visual progress toward a debt-free date
- Clean, minimal UI (no overwhelm)
- Motivation-focused experience
- Fast and lightweight

---

## 🧠 Why I Built This

I wanted something simple to track debt without spreadsheets or complicated apps.

Most tools made me feel overwhelmed instead of motivated, so I built Zeni to feel calm, clear, and encouraging.

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- TanStack Router
- Supabase (Auth + Database)
- Vercel (Deployment)

---

## 🔐 Environment Variables

Create a `.env` file in the root of the project:

VITE_SUPABASE_URL=your_url_here
VITE_SUPABASE_PUBLISHABLE_KEY=your_key_here
